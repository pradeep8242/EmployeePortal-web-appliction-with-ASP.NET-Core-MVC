# Employee Portal Web Application (ASP.NET Core MVC)

## 📌 Overview

Employee Portal is a web-based Employee Management System developed using ASP.NET Core MVC. The application allows users to manage employee records efficiently through Create, Read, Update, and Delete (CRUD) operations. It follows the MVC architecture and uses Entity Framework Core for database interaction with SQL Server.

## 🚀 Features

- Add New Employee
- View Employee Details
- Update Employee Information
- Delete Employee Records
- Employee Search
- Department Management
- Employee Type Management
- Form Validation
- Responsive User Interface
- SQL Server Database Integration
